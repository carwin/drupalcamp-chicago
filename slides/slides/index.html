Tools For Rapid Drupal Deployment



What is "Rapid Drupal Deployment" ?



  • Building re-usable platforms rather than
    building singular sites
  • Focusing on repeatable configurations to
    launch the same platform on multiple servers
  • Separating our configuration from our content
    (as much as possible).





WHY?

  1. It saves time.
  2. We like things DRY.
  3. It saves time.
  4. Re-usable is good.

Installation Profiles


What is an 
Installation Profile?


  • Configuration
  • Dependencies
  • Modifications
  • "Marching orders" for installation


What do we need to create one?


  • A .info file - to describe the various module
    dependencies you want to enforce.

  • A .install file - to set your variables and
    Configure All The Things â„¢

  • A .profile file - to use various Drupal hooks
    like hook_form_alter() or hook_image_default_styles()

Sample.info


name = Great Profile!
description = Really, this is better than other install profiles.
core = 7.x

;Core modules
dependencies[] = php
dependencies[] = rdf
dependencies[] = block

;Contrib modules
dependencies[] = pathauto
dependencies[] = menu_block
dependencies[] = token

Sample.install


function sample_install() {
    
    // Turn off comments for basic pages
    variable_set('comment_page', COMMENT_NODE_HIDDEN);
    
    // No user pictures
    variable_set('user_pictures', 0);
    
    // Theme stuff!
    variable_set('theme_default', 'bartik');
    variable_set('admin_theme', 'rubik');
    variable_set('node_admin_theme', '1');
    thme_enable('tao', 'rubik'));
}

Sample.profile


function sample_form_alter(&$form, $form_state, $form_id) {
  switch($form_id) {
    case "install_configure_form":
    $form['admin_account']['account']['name]['#default_value'] = 'admin'
    $form['admin_account']['account']['mail']['#default_value'] = 'me@me.com'
       break;
  }
}

Drush

(your first Drush experience)

Things Drush can do for us:


Download Drupal
> drush dl drupal OR drush dl drupal-7.13
Download modules
> drush dl contrib_module
Enable modules
> drush en contrib_module
Evaluate PHP
> drush ev "print 'woot';"

Makefiles

What Is a Makefile?

Manifest of resources.
Can download install profiles, themes, modules, libraries, and features.

Where Can Makefiles Download Their Resources From?

  • Drupal.org
  • Feature Servers (fserver)
  • Version Control
  • FTP
